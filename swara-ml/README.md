## Recipes
### Kernels
- Linear Kernel: $$K(x, y) = x^T y$$
- Polynomial Kernel: $$K(x, y) = (x^T y + c)^d$$
- Sigmoid Kernel: $$K(x, y) = tanh(\gamma x^T y + r), \gamma > 0$$
- Gaussian RBF Kernel: $$K(x, y) = exp(- \gamma \parallel x - y \parallel^2)$$
